# Farm Mini Game

A browser-based 3D farming mini game built with Three.js and PixiJS.

## Gameplay

- Plant crops and harvest them when fully grown
- Buy animals (chickens, cows) and collect their produce
- List items for sale at your market stall
- Customer cars drive up, buy your produce, and pay you coins
- Expand your farm grid as you earn more

## Tech Stack

- **Three.js** — 3D scene, models, lighting, shadows
- **PixiJS** — HUD, panels, modals (overlaid on the 3D canvas)
- **Howler.js** — Spatial audio and background music
- **Vite** — Dev server and bundler

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
